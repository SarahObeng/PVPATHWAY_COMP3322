# PVPATHWAY_COMP3322

Overview

The goal is to design a structured, maintainable course registration system

The database is developed using MySQL (XAMPP) and managed collaboratively through GitHub.

Repo Structure

pvamu_registration_system/
│
├── database/
│   ├── schema.sql
│   ├── seed_data.sql
│   └── migrations/
│
├── backend/
│
├── frontend/
│
└── README.md

Setup Instructions

Start XAMPP (Apache and MySQL).

Open a browser and go to:

http://localhost/phpmyadmin

Create a new database named:

pvamu_registration

Click Import.

Select database/schema.sql.

Execute the import.
